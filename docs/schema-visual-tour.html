§5

The stamp — approval built into the furniture

Every outbound message, publication and invoice climbs this ladder. It isn't a polite instruction to the AI — it's a rule of the database itself.

Draft

Light writes it. Nobody outside can see it.

→

Pending approval

Appears in your approval inbox, waiting.

→

Human stamp

A human approver's name is written onto the row. Light cannot forge this.

→

Sent

Only now does the send pipeline accept it.

The send pipeline physically refuses any message whose approver is not a human. This is the "AI COO, human CEO" promise expressed as a database constraint — the strongest form a promise can take.

§6

The ledger — nothing is ever erased

Underneath everything runs one append-only book of record. Every layer writes to it; no one — not even you — can edit or delete a line. This is the audit trail, and it is built first.
